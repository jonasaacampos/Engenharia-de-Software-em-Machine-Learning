## Definição de Aplicação Móvel

Aplicativo móvel (ou app móvel) é um software projetado para ser executado em um dispositivo móvel (smartphone ou tablet). 

Diferente dos programas desenvolvidos para computador, *aplicações móveis possuem funcionalidades isoladas e limitadas*.

Mesmo nos dispositivos mais modernos, os recursos de hardware de um dispositivo móvel e seus fabricantes desencorajam aplicativos multifunção.
